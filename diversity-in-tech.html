Project Brief

Diversity In Tech was a site I created to learn more about the lack of diversity in tech and also to use different tools to build out the site.

View the project

Tools Used

  • HTML5/CSS3/SASS
  • Burbon/Neat/Bitters/Refill (Frameworks)
  • Javascript/JQuery
  • Responsive Web Design
  • Sketch ( Web Design )

Process

It took me two weeks to complete this project, with research and competitive market analysis, user surveys, style tiles, wireframes, mockups, and using Illustrator to create all of the logos and icons for the site, before diving into production in code. Check out out this Style Guide for more details on my process and design elements!
